#### 使用说明

1. 工具安装：在您的电脑上安装Git for Windows软件

2. 仓库下载：使用Git clone命令将仓库所有文档下载到本地电脑：

   ```bash
   # git clone https://gitee.com/wang_litao/doc.git
   ```

3. 实时更新：使用 git pull 命令，可以将远程仓库最新版本的文档、代码同步到本地电脑

   ```bash
   # cd doc
   # git pull
   ```
